About

This project is an integration of three different aquaponic forums where user can find relevant data on different aquaponic discussions based on his/her search. Here we pull all the data from three different forums in to one interface starting from recent activity for the search requested. As the discussions on different topics change every single day.

The outcome of the Aquaponic Forum Hub would be the results which help the aquaponic gardeners who can find efficient solutions from many sources at one place to improve their productivity. The user can also share the search results on social networking websites which makes the website much interesting.

We would be integrating the data from following sources:

• The Aquaponic Source
• Aquaponics Nation
• Backyard Aquaponics
